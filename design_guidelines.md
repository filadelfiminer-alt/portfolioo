# Дизайн-Гайдлайны: Сайт-Портфолио

## Подход к Дизайну

**Референсы**: Behance (галерея проектов), Dribbble (карточки работ), Notion (admin панель)

Создаем визуально впечатляющее портфолио с двумя контрастными интерфейсами:
- **Публичная часть**: Эмоциональная, визуально-богатая презентация работ
- **Админ-панель**: Чистая, утилитарная система управления контентом

## Типографика

**Шрифты** (Google Fonts):
- **Заголовки**: Inter (700, 800) - современный, четкий
- **Основной текст**: Inter (400, 500)

**Иерархия**:
- Hero заголовок: text-6xl md:text-7xl lg:text-8xl font-bold
- Заголовки секций: text-4xl md:text-5xl font-bold
- Названия проектов: text-2xl md:text-3xl font-bold
- Описания: text-lg md:text-xl
- Метаданные: text-sm font-medium

## Система Отступов

Используем Tailwind единицы: **4, 6, 8, 12, 16, 20, 24**
- Внутренние отступы компонентов: p-6, p-8
- Расстояния между секциями: gap-8, gap-12, gap-16
- Внешние отступы секций: py-16 md:py-24 lg:py-32

## Публичная Галерея

### Структура Страницы

**1. Hero Секция** (100vh)
- Полноэкранный визуальный impactный момент
- Крупная типографика с именем/слоганом
- Размытый backdrop для кнопки CTA (если поверх изображения)
- Использовать яркое, профессиональное hero-изображение

**2. Галерея Проектов** (Masonry Grid)
- Асимметричная раскладка в стиле Pinterest/Behance
- Колонки: grid-cols-1 md:grid-cols-2 lg:grid-cols-3
- Проекты разной высоты для динамики
- Hover эффект: легкое увеличение (scale-105) + тень

**3. Карточки Проектов**
Каждая карточка содержит:
- Изображение превью проекта (aspect-video или aspect-square)
- Название проекта (крупное, жирное)
- Краткое описание (2-3 строки)
- Теги/категории (rounded-full badges)
- Overlay при hover с призывом "Смотреть проект"

**4. Фильтр/Навигация**
- Sticky горизонтальная панель тегов над галереей
- Активные теги с визуальным выделением
- Плавная анимация фильтрации (fade in/out)

**5. Модальное Окно Проекта**
При клике открывается полноэкранная модалка:
- Крупные изображения проекта (карусель/слайдер)
- Подробное описание справа
- Технологии, роль, дата
- Внешние ссылки (GitHub, Live Demo)
- Кнопка закрытия (фиксированная)

## Админ-Панель

### Структура

**Боковая Навигация** (фиксированная, слева):
- Логотип/имя вверху
- Пункты меню: Проекты, Загрузить, Настройки
- Индикатор авторизации внизу

**Основная Область**:
- Заголовок страницы с breadcrumbs
- Таблица проектов или форма загрузки
- Минималистичный, функциональный дизайн

**Форма Загрузки**:
- Drag-and-drop зона для изображений (большая, с пунктирной рамкой)
- Превью загруженных изображений (grid с возможностью удаления)
- Поля: Название, Описание (textarea), Теги (multi-select), Ссылки
- Крупная кнопка "Опубликовать" справа

## Компоненты

**Кнопки**:
- Основная CTA: px-8 py-4 rounded-lg font-semibold
- Вторичная: px-6 py-3 rounded-lg
- Размытый backdrop для кнопок поверх изображений

**Карточки**:
- rounded-xl с тонкой тенью
- Padding: p-6 или p-8
- Hover: transform transition-all duration-300

**Инпуты (Админ)**:
- Чистые, с минимальными рамками
- Focus state: четкое выделение
- Labels: text-sm font-medium сверху

**Badges/Теги**:
- rounded-full px-4 py-2
- Небольшие, аккуратные
- Кликабельные в галерее для фильтрации

## Иконки

**Heroicons** (через CDN) - outline для основного интерфейса, solid для акцентов

## Изображения

**Hero**: Профессиональное, атмосферное изображение (рабочее пространство, абстракция, или лучшая работа)
**Проекты**: Скриншоты/mockups работ, качественные превью
**Админ**: Иконки drag-and-drop зоны

## Анимации

Использовать минимально:
- Hover эффекты на карточках (scale, shadow)
- Плавное появление модального окна
- Fade transitions при фильтрации
- **Без** сложных scroll-анимаций или parallax

## Адаптивность

- Mobile-first подход
- Галерея: 1 колонка на mobile, 2 на tablet, 3 на desktop
- Модалка: fullscreen на mobile, centered с max-width на desktop
- Админ: боковая навигация сворачивается в hamburger на mobile
